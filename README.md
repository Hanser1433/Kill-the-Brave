# 🗡️ Kill The Barve（开发中）  
_「当勇者开始怀疑自己的剧本，弑神就成了唯一的救赎」_  

## 🎮 核心体验
- ​**动态人格系统**：选择影响角色属性与剧情分支  
- ​**哲学谜题**：自由意志、电车难题等命题抉择  
- ​**多结局网络**：基于`自我认知值`解锁9种结局  
- ​**颠覆性叙事**：解构传统勇者拯救世界的套路

## 🛠️ 技术栈
| 模块         | 工具                          |
|--------------|------------------------------|
| 游戏引擎     | Unity 2022                   |
| 叙事系统     | Yarn Spinner + 自定义分支树   |
| 像素美术     | Aseprite + Pyxel Edit        |
| 音效设计     | Bosca Ceoil + Audacity       |
| 版本控制     | Git + GitHub Projects        |
